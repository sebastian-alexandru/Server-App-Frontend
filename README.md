# Server-App-Frontend

Frontend App for https://github.com/sebastian-alexandru/Server-App-Backend

CRUD application for managing servers and checking their status using:

DB: PostgreSQL

ORM: Hibernate

BE: SpringBoot

FE: Angular, RxJS

Containerization: Docker, Kubernetes
